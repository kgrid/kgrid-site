# Integrator's Guide

This guide describes how to run and configure Knowledge Grid components for your environment.

The KGrid Activator (opens new window) and KGrid Library (opens new window) components are Spring Boot (opens new window) java applications.

For running the application you need:

  • Java 11 or higher
    • Oracle Java (opens new window)
    • OpenJDK (opens new window)
